# RME 声卡残响修复工具 使用说明

## 1. 概述

RME 声卡残响修复工具是一款系统托盘中运行的无界面程序，用于通过播放特定音频信号来修复 RME 声卡的残响问题。

**工作流程：**
1. 启动时自动播放「1-开始.wav」
2. 触发修复操作时，向**每一块声卡**同时播放「2-静音.wav」
3. 所有声卡播放完毕后，播放「3-完成.wav」提示

## 2. 系统要求

- Windows 10 64 位或更高版本

## 3. 文件结构

| 文件/目录 | 说明 |
|-----------|------|
| `RMEFixer.exe` | 主程序 |
| `config.json` | 快捷键配置文件（首次运行自动生成） |
| `sound/` | 音频文件目录 |

## 4. 首次运行

双击 `RMEFixer.exe`，通知区域出现蓝色喇叭图标，自动播放「1-开始.wav」。

如需退出，点击图标 → 托盘菜单选「退出」，或直接关闭进程。

## 5. 功能说明

### 5.1 快捷键

默认快捷键：`Ctrl + Win + Alt + F10`

按下快捷键触发**修复残响**操作：
1. 系统遍历所有声卡设备
2. 每张声卡同步播放「2-静音.wav」
3. 全部播放完毕后播放「3-完成.wav」

### 5.2 托盘菜单

左键单击 / 右键单击 / 回车 / Application 键均可弹出菜单：

- **修复残响** — 执行与快捷键相同的操作
- **退出** — 退出程序

## 6. 自定义快捷键

编辑 `config.json`（与 `RMEFixer.exe` 同目录），修改 `hotkey` 字段：

```json
{
  "hotkey": "Ctrl+Win+Alt+F10"
}
```

**修饰键**（支持组合，用 `+` 连接）：

| 键 | 说明 |
|----|------|
| `Ctrl` | 控制键 |
| `Alt` | 替代键 |
| `Win` | Windows 键 |
| `Shift` | 上档键 |

**主键**（修饰键之后最后一位）：

| 类别 | 支持值 |
|------|--------|
| 功能键 | `F1` ~ `F24` |
| 字母键 | `A` ~ `Z` |
| 数字键 | `0` ~ `9` |
| 其他 | `Space`、`Enter`、`Tab`、`Esc`、`Backspace`、`Delete`、`Home`、`End`、`PageUp`、`PageDown`、方向键等 |

**示例：**

```json
{ "hotkey": "Ctrl+Alt+F9" }
```

如果配置无法解析，程序会自动回退到默认快捷键 `Ctrl+Win+Alt+F10`，不会崩溃。

## 7. 多开处理

程序启动时会自动检测并关闭已有的旧进程，确保同一时间只有一个实例运行。

## 8. 常见问题

**Q: 快捷键没反应？**
A: 检查是否与其他软件热键冲突，尝试在 `config.json` 中更换一组快捷键。

**Q: 无法播放声音？**
A: 确保 `sound/` 目录下的 `.wav` 文件存在且完整。检查系统音量设置。

**Q: 程序启动后托盘没有图标？**
A: 检查 Windows 通知区域设置，确保允许 `RMEFixer.exe` 显示图标。

**Q: 所有声卡播放是什么意思？**
A: 程序会枚举系统中的每一块物理音频输出设备（包括 USB 声卡、HDMI 音频等），同时对每一块设备播放静音音频，用于清除残响缓冲。
